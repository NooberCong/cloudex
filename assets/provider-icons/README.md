# Provider Icons

Place your custom provider icons in this folder using these exact basenames:

- aws-s3
- cloudflare-r2
- backblaze-b2
- wasabi-s3
- minio-s3
- digitalocean-spaces

Accepted extensions (fallback order): `.svg`, `.png`, `.webp`, `.jpg`, `.jpeg`, `.gif`

Examples:
- `aws-s3.webp`
- `cloudflare-r2.png`
- `backblaze-b2.svg`

These basenames match `ProviderType` values in `src/types/index.ts`.
